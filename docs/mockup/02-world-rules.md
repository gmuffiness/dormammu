> **분기점:** 시가시나 탈환 작전 (850년)

## 1. 분기점 직전 정치/군사 상황

| 항목 | 상태 |
|------|------|
| 파라디 섬 내부 | 히스토리아 레이스 즉위(여왕), 쿠데타 성공 후 조사병단이 실권 장악 |
| 마레와의 관계 | 마레는 시조 거인 탈환을 장기 목표로 설정. 시가시나 탈환 시점에는 아직 직접 접촉 없음 |
| 9거인 보유 현황 | 마레: 초대형/갑옷/여자/턱/짐승/수레 (6기). 파라디: 시조+진격(에렌) |
| 벽 바깥 정보 | 시가시나 지하실 열람 전까지 미지. 탈환작전 중 그리샤 일지 발견으로 처음 공개 |
| 중동연합-마레 전쟁 | 시가시나 탈환 시점 이미 진행 중. 마레의 거인 군사력 의존도 노출 |

## 2. 거인의 힘 규칙

### 2-1. 유미르의 저주 (Curse of Ymir)
- **규칙:** 9거인 계승자는 계승 시점부터 정확히 13년 후 사망
- **카테고리:** magic
- **분기 영향:** 엘빈이 초대형 거인을 계승하면 13년 카운트다운 시작. 나이(~35-38세) 고려 시 48-51세에 사망
- **출처:** [Power of the Titans - AoT Wiki](https://attackontitan.fandom.com/wiki/Power_of_the_Titans)

### 2-2. 9거인 계승 규칙
- **규칙:** 순수 거인이 9거인 계승자를 섭취하면 능력 계승 + 인간 복귀
- **카테고리:** magic
- **분기 영향:** 엘빈에게도 동일 적용 (세럼 → 거인화 → 베르톨트 섭취)
- **출처:** [Power of the Titans - AoT Wiki](https://attackontitan.fandom.com/wiki/Power_of_the_Titans)

### 2-3. 시조 거인 (Founding Titan)
- **규칙:** 왕가(프리츠/레이스) 혈통만 완전 능력 개방 가능. 유미르의 백성 전체 기억 조작, 순수 거인 조종, 육체 개조
- **제약:** 에렌은 왕가 혈통 아님 → 지크(왕가 혈통)와의 접촉으로 우회 사용
- **카테고리:** magic/politics
- **분기 영향:** 불변 — 엘빈 생존 여부와 무관
- **출처:** [Ymir Fritz - AoT Wiki](https://attackontitan.fandom.com/wiki/Ymir_Fritz)

### 2-4. 진격의 거인 — 미래 기억
- **규칙:** 계승자는 후계자들의 미래 기억을 역방향으로 수신 가능 (단편적, 제어 불가)
- **핵심 메커니즘:** 에렌 크루거 → 그리샤 → 에렌의 인과 루프. "미카사와 아르민을 지켜라"
- **카테고리:** magic/physics (시간 인과율)
- **분기 영향:** 에렌이 계승자로 유지되므로 능력은 불변. 단, 엘빈이 에렌의 미래 기억 기반 행동을 어디까지 인지/대응할 수 있는가가 핵심 쟁점
- **출처:** [Attack Titan Future-Seeing Ability - Game Rant](https://gamerant.com/attack-on-titan-attack-titan-future-seeing-ability-explained/)

## 3. 아커만 가문 특수 규칙

- **규칙:** 거인 과학의 부산물로 생성된 강화 인간. 왕가의 기억 조작 및 시조 거인의 정신 지배 완전 면역
- **각성 조건:** '숙주(Host)' 설정 + 죽음 직면 시 각성. 역대 아커만 전투 경험을 경로(Paths)로 계승
- **해당 인물:** 리바이, 미카사
- **카테고리:** magic
- **분기 영향:** 불변
- **출처:** [Ackerman Clan - AoT Wiki](https://attackontitan.fandom.com/wiki/Ackerman_clan)
