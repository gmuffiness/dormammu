## Narrative
직접 대면 없이 시작되는 정보전. 지크는 파라디 내부 접선원을 통해 엘빈의 행동을 추적하고, 엘빈은 마레에서 흘러나오는 정보를 통해 지크의 패턴을 역추적한다. 두 사람은 상대의 존재를 인식하고, 서로를 체스판의 상대로 인식한다.

## Key Events
- 엘빈, 마레의 첩보 패턴 분석 중 "내부에서 이견이 있다 — 마레의 단일 대표가 아닌 독립 변수"를 추론
- 지크, 엘빈이 자신을 역추적하고 있다는 신호를 감지 — 처음으로 상대를 진지하게 평가
- 둘 다 상대를 "설득 불가능한 적"이 아니라 "이해해야 할 변수"로 보기 시작
- 복선: 엘빈의 메모 — "지크 예거 — 에렌의 이복형. 동기 불명. 마레 전략과 일치하지 않는 부분이 있다." 여백에 물음표 하나
