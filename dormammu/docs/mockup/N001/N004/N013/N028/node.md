## Narrative
마레 첩보부가 파라디 내부에 이미 잠복시킨 요원들을 활성화하기 시작한다. 목표: 엘빈과 에렌 사이의 균열을 확대하고, 파라디 내부 여론을 "조사병단 독점 권력"에 반발하도록 유도.

## Key Events
- 마레 요원, 파라디 시민들 사이에 "조사병단이 진실을 독점한다"는 소문 유포
- 마레 요원, 에렌에게 익명의 메시지를 전달 시도 — "당신의 사령관은 당신을 도구로 쓰고 있다"
- 한지: "이건 자연발생적인 불만이 아니야." 엘빈에게 보고
- 리바이: "첩자를 잡아야 한다." 엘빈: "잡는 것보다 역이용이 먼저다."
- 복선: 에렌이 버린 익명 메시지의 구절이 자꾸 떠오른다. 엘빈에게 아르민의 죽음에 대해 직접 물어보고 싶다는 충동
